### 第三步：AI内容生成（自动化核心）

#### 1. 产品标题与描述
**ChatGPT Prompt模板**：
```
你是一个专业的跨境电商产品描述写手。请为以下产品写吸引人的标题和描述：

产品：星座主题手机壁纸（12星座）
特点：AI生成，高清，每月更新
目标客户：18-30岁女性，喜欢星座和美学
售价：$8.99
平台：Etsy

要求：
1. 标题（包含关键词）
2. 短描述（80字内）
3. 长描述（分点说明优势）
4. 标签（10个相关标签）
5. 常见问题（3个）

请用英语写作，风格亲切专业。
```

#### 2. 产品图片
**Midjourney Prompt模板**：
```
product photo of [产品名], clean background, studio lighting, professional ecommerce, high detail, 8k --ar 3:4 --style raw
```

**批量生成技巧**：
- 一次生成10-20个变体
- 挑选最好的3-5个
- 用Canva加文字和Logo

#### 3. 客服回复
**ChatGPT客服模板**：
```
你是一个跨境电商客服。请回复客户关于发货时间的问题：

客户问题：How long does shipping take to China?
产品：数字产品（即时下载）
实际情况：购买后立即收到下载链接

请用友好专业的语气回复，包含：
1. 明确答案
2. 额外帮助（如有）
3. 感谢用语
```

### 第四步：上架与定价

#### 上架清单：
1. **主图**：3-5张高质量图（不同角度/场景）
2. **标题**：包含主要关键词
3. **描述**：分点说明，突出优势
4. **价格**：测试不同价位
5. **标签**：10-13个相关标签
6. **变体**：不同颜色/尺寸（如有）

#### 定价策略：
```
成本定价法：
数字产品：成本×5-10倍
实物产品：（成本+运费）×2.5-3倍

心理定价：
$9.99比$10好卖
套餐价：3个$24.99（原价$29.97）

测试方法：
第1周：$12.99
第2周：$14.99（如果销量稳定）
第3周：$9.99（促销测试）
```
