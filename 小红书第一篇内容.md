### 🎯 第一步：找到爆款模板
先分析10个同领域爆款笔记，总结出这3个万能模板：

1. **痛点解决型**
   - 开头：你是不是也遇到XXX问题？
   - 中间：3个解决方法（ChatGPT生成）
   - 结尾：行动起来！

2. **案例分享型**  
   - 开头：我朋友/学员通过XXX赚了XX元
   - 中间：具体操作步骤
   - 结尾：你也可以复制！

3. **工具推荐型**
   - 开头：发现一个神仙AI工具
   - 中间：具体使用教程
   - 结尾：免费获取方式

### 🤖 第二步：用ChatGPT批量生成
给ChatGPT这个提示词，一次生成10个不同角度的文案：

```
你是一个小红书爆款文案专家，请根据以下要求生成文案：

主题：AI副业赚钱
目标受众：25-35岁女性，想找副业
风格：亲切、实用、有获得感
长度：300-500字

请生成3个不同角度的文案：
1. 痛点解决型（解决"没时间做副业"问题）
2. 案例分享型（分享成功案例）
3. 工具推荐型（推荐AI工具）

每个文案包含：
- 吸引人的标题
- 3个核心要点
- 行动号召
- 相关话题标签
```

### 📊 第三步：优化和发布
ChatGPT生成的文案需要人工优化：
1. **加emoji**：每段加1-2个相关表情
2. **分段**：2-3行一段，阅读不累
3. **加话题**：#AI赚钱 #副业 #ChatGPT #女性成长
4. **互动引导**：结尾问问题，增加评论
