# 小红书封面图设计指南

## 封面图主题
**用ChatGPT做小红书爆款文案**

## 设计元素
1. **左侧**：ChatGPT对话界面
   - 显示提示词输入
   - 显示AI回复的文案
   - 科技感、蓝色主题

2. **右侧**：小红书爆款文案展示
   - 高点赞、高收藏的笔记截图
   - 显示数据：5000+点赞，200+评论
   - 小红书经典红色元素

3. **中间连接**：
   - 箭头从ChatGPT指向小红书
   - 文字："AI生成 → 爆款文案"
   - 发光效果

4. **标题文字**：
   - 主标题：🔥 我用ChatGPT一个月写出100+爆款文案
   - 副标题：方法全公开！小白也能学会
   - 字体：粗体，醒目颜色

5. **装饰元素**：
   - 星星、闪光效果
   - 小红书logo元素
   - AI、机器人小图标

## 颜色方案
- **主色**：小红书红 (#FF2442) + ChatGPT蓝 (#10A37F)
- **背景**：白色或浅灰色渐变
- **文字**：黑色或深灰色，标题用红色/蓝色

## 尺寸要求
- **小红书封面图**：3:4比例
- **推荐尺寸**：1080px × 1440px
- **安全边距**：留出50px边距

## 在线制作工具推荐
1. **Canva**（最推荐）：
   - 模板丰富，操作简单
   - 有小红书专用模板
   - 免费版足够使用

2. **稿定设计**：
   - 中文界面友好
   - 小红书模板多

3. **Figma**：
   - 更专业的设计工具
   - 需要一些设计基础

## Canva制作步骤
1. 访问 https://www.canva.com
2. 搜索"小红书封面"
3. 选择科技/教育类模板
4. 修改文字内容
5. 调整颜色匹配主题
6. 导出为PNG
